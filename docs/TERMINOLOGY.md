# Buyer Lite — Терминология

## Бизнес-термины

### Скрытая маржа (Hidden Margin)

Общий термин для всей "невидимой" выгоды байера. Делится на два типа:

#### 1. Курсовой спред (Exchange Spread)

Разница между реальным курсом обмена и курсом, который байер называет клиенту.

```
Пример:
- Клиент отправил 100 000 RUB
- Байер обменял по курсу 1.17 (реальный)
- Клиенту сказал курс 1.16
- Спред: (1.17 - 1.16) × 100 000 = 1 000 KGS
```

**В системе:**
- `rate_real` — реальный курс (для внутреннего учёта)
- `rate_client` — курс для клиента (показывается в отчёте)

#### 2. Товарная маржа (Product Margin)

Разница между реальной ценой закупки и ценой, которую байер называет клиенту.

```
Пример:
- 100 блузок закуплено по 115 сом/шт (реально)
- Клиенту сказано 120 сом/шт
- Маржа: (120 - 115) × 100 = 500 KGS
```

**В системе:**
- `purchase_price` — реальная цена закупки (для внутреннего учёта)
- `client_price` — цена для клиента (показывается в отчёте)

---

### Комиссия байера (Buyer Commission)

Фиксированные **5%** от стоимости товаров (по цене для клиента).

```
Пример:
- Товары для клиента: 12 000 KGS (по цене для клиента)
- Комиссия: 12 000 × 5% = 600 KGS
```

**В системе:** автоматически рассчитывается, отображается в отчёте.

---

### Баланс клиента (Client Balance)

Доступные средства клиента в системе (в KGS).

```
Баланс = (пополнения) 
       - (стоимость товаров по цене для клиента) 
       - (доставка) 
       - (комиссия 5%)
```

**В системе:**
- `balance` — текущий баланс
- `balance_transactions` — история всех операций

## Сущности

| Русский | English | Описание |
|---------|---------|----------|
| Байер | Buyer | Пользователь системы |
| Клиент | Client | Получатель товаров из России |
| Поставщик / Точка | Supplier | Место на Дордое |
| Заказ | Order | Закупка у поставщика |
| Товар | Order Item | Конкретная позиция в заказе |
| Отправка | Shipment | Группа товаров для отправки через карго |
| Операция обмена | Exchange Operation | Обмен валюты |
| Отчёт | Report | PDF-документ для клиента |

---

## Статусы

### Заказ (Order)

| Статус | Описание |
|--------|----------|
| `planned` | Запланирован, ещё не закуплен |
| `purchased` | Закуплен у поставщика |
| `ready` | Готов к отправке |
| `shipped` | Отправлен через карго |

### Отправка (Shipment)

| Статус | Описание |
|--------|----------|
| `preparing` | Формируется, товары подобраны |
| `shipped` | Отправлена через карго |
| `delivered` | Доставлена клиенту |
